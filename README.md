This repository contains the code for the optimization algorithms of the Hybrid Optimization Methodologies for Transportation Logistics Management – HybOpt. 
This research project (proposal number: 02562) is funded by the Hellenic Foundation for Research & Innovation (H.F.R.I.) through the “2nd Call for H.F.R.I. Research Projects to support Faculty Members and Researchers” program. 
The folders contain the implementation for solving the following problems:

1. Cyclic Production Routing Problem
2. Set Orienteering Problem 
3. Set Team Orienteering Problem
4. Cumulative Capacitated Vehicle Routing Problem
5. Vehicle Routing Problem with Delivery Options 
6. Crowdsourced Humanitarian Relief Vehicle Routing Problem
7. Family Capacitated Vehicle Routing Problem 

The implementation is in C#, with some usage of Python.
